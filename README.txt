####Version 1.0#####

node.js script which handles requests for basic vehicle controls. Currently needs actual communication code to a module. 
Input:
Voice related to vehicle controls. Script can take natural commands as well as seat position/door location.
	--> Turn on/Turn off Car
	--> Open/Close Window
	--> Lock/Unlock Door
	
Output: 
	--> Corresponding voice response to input.
	Example: "Open Driver Window = Opening Driver Window"
